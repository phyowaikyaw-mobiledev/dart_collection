## ğŸš€ Getting Started

### Prerequisites

* [Dart SDK](https://dart.dev/get-dart) 2.19 or higher

### Installation

1. **Clone the repository:**
```bash
git clone https://github.com/phyowaikyaw-mobiledev/dart_collection.git
cd dart_collection
```

2. **Install dependencies:**
```bash
dart pub get
```

3. **Run examples:**
```bash
# Run main file
dart run main.dart

# Run with hot reload (if using dartpad)
dart run --enable-vm-service
```
